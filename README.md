# amkyaw.dev - Web Development Platform

A modern web development platform with AI integration, code editor, and cloud storage.

## Features

- 🎨 **3D Animated Landing Page** with Three.js
- 🔐 **Firebase Authentication** for secure login
- 🤖 **AI Chat Assistant** powered by Gemini AI
- 💻 **Online Code Editor** with multiple language support
- 📁 **Google Drive Integration** for project storage
- 🌙 **Dark/Light Mode** with smooth transitions
- 🌍 **Multi-language Support** (English, Myanmar, Thai, Chinese, Russian)
- 📱 **Responsive Design** for mobile and desktop
- 🎯 **Project Management** with file explorer

## Tech Stack

- HTML5, CSS3, JavaScript
- Tailwind CSS for styling
- Firebase for authentication
- Google Drive API for storage
- CodeMirror for code editing
- Three.js for 3D animations
- Cloudflare Worker for AI integration

## Project Structure

```

amkyaw.dev/
├── index.html              # Landing page with 3D animation
├── auth/                   # Authentication pages
├── pages/                  # Main application pages
├── js/                     # JavaScript files
├── css/                    # Stylesheets
├── components/             # Reusable components
├── img/                    # Images and icons
├── lug/                    # Language files
├── server/                 # Server-side code
└── worker.js              # Cloudflare Worker

```

## Setup Instructions

1. Clone the repository
2. Set up Firebase project and add config
3. Configure Google Drive API
4. Deploy Cloudflare Worker with Gemini API key
5. Update API keys in configuration files

## Usage

- **Demo Mode**: 3 free attempts without login
- **Full Access**: Login with Firebase account
- **Create Projects**: Use the coder to build web applications
- **AI Assistance**: Chat with AI for coding help
- **Save Projects**: Store projects in Google Drive
